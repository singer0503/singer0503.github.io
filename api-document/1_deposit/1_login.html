    # 玩家登入(Login)

    TIP

    可请商务提出需要 API 测试集成范例，Ending Game 提供 Postman 的汇入档，缩短您的开发时间

    # Endpoint

    POST http://159.138.33.136:8081/api/business/merchant/login

    # Parameters

    Header
    Content-Type application/json
    Body Parameters Desc Required Type
    token 商戶的 token * string
    player_account 玩家帳號 * string
    deposit 需要轉入的金額，可為 0 * int

    # Request Body Sample

    {
    "token":"******9972d12663******462c8f3d28a******f8ea56e00392becf4b*******",
    "player_account":"maxtestplayer1",
    "deposit": 100
}

    # 盤口轉出全部金額(Withdrawal)

    # Endpoint

    POST https://mobile.ending.best/api/business/merchant/withdrawal

    # Parameters

    Header
    Content-Type application/json
    Body Parameters Desc Required Type
    token 商戶的 token * string
    player_account 玩家帳號 * string

    # Request Body Sample

    {
    "token":"******9972d12663******462c8f3d28a******f8ea56e00392becf4b*******",
    "player_account":"maxtestplayer1"
}

    # Response Body Sample

    成功返回结果 回傳玩家當前的帳號以及遊戲內的餘額，以及轉出的金額

    Status Code: 200 OK
{
    "account": "maxtestplayer1",  // 玩家帳號
    "balance": 0,                 // 玩家遊戲內的餘額
    "out_cash": 200               // 轉出的金額
}

    成功, 但玩家游戏内没有金额

    Status Code: 200 OK
{
    "account": "maxtestplayer1",
    "balance": 0,
    "out_cash": 0
}

    失败返回结果

    Status Code: 500 Internal serrver error
{
    "code": "UserNotExists",
    "msg": "token is not exists"
}

    TIP

    该接口最长响应时间最长30秒，请在请求时配置好接口超时时间。

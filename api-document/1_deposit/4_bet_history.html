    # 下注记录查询(Bet History)

    # Endpoint

    POST https://mobile.ending.best/api/business/merchant/bet_history

    # Parameters

    Header
    Content-Type application/json
    Body Parameters Desc Required Type
    token 商戶的 token * string
    player_account 玩家帳號 string
    bet_type 注单状态 int
    page_size 每页数量 （最大 100 条数据, 预设 50 条数据） int
    page 第几页 （预设第 1 页） int

    # Request Body Sample 1

    查询全部玩家下注数据

    {
    "token":"******9972d12663******462c8f3d28a******f8ea56e00392becf4b*******"
}

    # Request Body Sample 2

    查询单一玩家数据

    {
    "token":"******9972d12663******462c8f3d28a******f8ea56e00392becf4b*******",
    "player_account":"maxtestplayer1"
}

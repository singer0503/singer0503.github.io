# 🏀 Ending Game

Ending Game 源自于 "宇博科技" 是行业的先行者，深耕行业游戏和盤口服务。自公司成立以来，宇博科技秉承成就客户、极致服务、实事求是、开放分享的核心价值观，以游戏服务改变生活为使命，致力成为行业一流的游戏盤口服务平台。

公司在成立初期就专注于游戏和盤口，是一家从事创新游戏产品研发，提供游戏科技综合解决方案的创新型科技公司。目前，公司规模超过 300 人，其中工程技术人员占比达 80%，公司核心团队来自国内外领先的互联网及软件技术企业。公司提供综合盘、反波胆、游戏等多种互联网支付产品，并已形成多个成熟的解决方案。
